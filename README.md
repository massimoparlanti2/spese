# 💸 Spese

Tracker di spese personali — Progressive Web App installabile su smartphone.

## Installazione su smartphone

### Android
1. Apri il sito in **Chrome**
2. Tocca i tre puntini (menu) → **"Aggiungi a schermata Home"**
3. Conferma → l'icona apparirà come un'app normale

### iPhone / iPad
1. Apri il sito in **Safari**
2. Tocca il tasto **Condividi** (rettangolo con freccia su)
3. Scorri e tocca **"Aggiungi a schermata Home"**
4. Conferma

## Funzionalità
- Registrazione spese con categorie ed emoji personalizzabili
- Riepilogo mensile e annuale
- Grafici e statistiche
- Budget per categoria
- Funziona offline (PWA)
- I dati sono salvati localmente sul dispositivo
